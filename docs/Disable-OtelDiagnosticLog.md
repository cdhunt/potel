# Disable-OtelDiagnosticLog

Remove the `OTEL_DIAGNOSTICS.json` file to disabled the internal logs generated by all OpenTelemetry components.

## Parameters


## Examples

### Example 1

Disable the internal logs.

```powershell
Disable-OtelDiagnosticLog
```
