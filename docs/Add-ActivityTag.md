# Add-ActivityTag

Add key-value data called Tags to an Activity.

## Parameters

### Parameter Set 1

- `[Hashtable]` **Tags** _A collection of key/value pairs._ Mandatory
- `[Activity]` **Activity** _An instance of an Activity._ Mandatory, ValueFromPipeline
